
# Task 8 - Simple Sales Dashboard

## Objective

To create an interactive sales dashboard showing performance by Month,
Region, and Category.

## Tools Used

-   Python (Pandas, Matplotlib)
-   ReportLab (PDF generation)

## Steps Performed

1.  Created a sample Superstore-like dataset.
2.  Converted Order Date into Month-Year format.
3.  Built 3 visuals:
    -   Line Chart: Sales over Months
    -   Bar Chart: Sales by Region
    -   Donut Chart: Sales by Category
4.  Summarized key insights from the dashboard.

## Insights

-   West region generated the highest sales.
-   Technology category had the highest contribution.
-   Sales peaked during mid-year months.
-   Clear monthly trends were visible.

## Outcome

Learned how to visualize sales data and present insights in dashboard
format.
